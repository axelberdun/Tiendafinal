 Placas de Video
Buenas, Este es mi proyecto. Armé una página web para vender placas de video

🛠️ Armado
HTML: Para armar el esqueleto de la página y acomodar los textos.
CSS: Para darle color, armar el diseño premium del menú y que las fotos no se rompan.
Google Fonts: Para cambiar la letra por una más linda (use la fuente Roboto).
Formspree: Para que el formulario de contacto funcione de verdad y los mensajes me lleguen al mail. Sume tambien el formspree y use pages
📐 Organización
El Menú: Quedó fijo arriba con un fondo oscuro. Si bajas en la página, el menú queda inmovilizado.
Los Productos: Puse tres placas de video con sus fotos y precios. Use Flexbox para que si entrás desde otro dispositivo se pueda ver bien
Las Reseñas: Los comentarios de los clientes los acomodan. Se adaptan tambien
Contacto: El formulario para que me manden consultas, vinculado con Formspree, para ver si me llega alguna consulta por mail
En línea: Ya lo subí a GitHub Pages y quedó la página andando en el hosting de Git con HTTPS
Se agrego el JSON para el local storage Estilo tipo tienda para poder comprar, acceder al sitio para loguearse y registrarse.
